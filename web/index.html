Intro to the Azure Lab

This lab is deployed using Azure App Services, blob storage, Azure functions, Azure DNS and more. To fully embrace the cloud and proper devops practices, I decided to deploy this entire app/site with Terraform. I use a CI/CD pipeline connected to my Github to deploy the infrastructure. It's cost-efficient (< $30/month) and overengineered just enough to simulate a production-grade environment.

This project served as a hands on learning ground for me to get into Terraform for the first time in my career. Learning how infrastructure as code works, how to deploy resources in Azure, and how to use Azure App Services to host a website was an adventure but eventually it came online.

Click on "About the Lab" to get a detailed breakdown of the infrastructure and how it all works. Note: This is all still a work in progress, so expect changes and improvements over time. I plan to add more features and content as I continue to learn and grow in my Azure journey.
